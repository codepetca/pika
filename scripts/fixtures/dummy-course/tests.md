Which keyword declares a value that cannot be reassigned?
Options:
- var
- let
- const
Correct Option: 3

### Question 2
Type: open_response
Points: 5
Prompt:
Explain the difference between `let` and `const` in your own words.
Answer Key:
`let` allows reassignment; `const` binds the name to one value.

### Question 2
Type: open_response
Points: 5
Prompt:

Answer Key:
