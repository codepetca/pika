# Pika Web Marketing Asset Notes

## Audience + Tone
- Audience: teachers, school admins, and students evaluating classroom workflow tools.
- Tone: modern, playful, practical.

## Core Message
Pika Classroom simplifies daily check-ins, assignments, and feedback so online and in-person classes stay aligned with less admin friction.

## Recommended Website Placements
1. Hero: `video/pika-walkthrough-60s.mp4` + poster `video/pika-walkthrough-poster.png`
2. Feature cards: selected stills from `screens/`
3. Captions: use `captions/pika-voiceover-60s.srt` for accessibility
4. Alternate format fallback: `video/pika-walkthrough-60s.webm`
